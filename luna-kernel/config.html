📖 Como usar a Luna Kernel

A Luna é uma agente autônoma que roda no seu PC (Linux). Ela conversa pelo Telegram e executa tarefas reais: cria projetos, edita arquivos, navega na web, gerencia o dashboard da NEXO DIGITAL. Todos no grupo usam a mesma conta — quando um pedir algo, a Luna executa no PC do Abner.

1

Iniciar a Luna

Via terminal, na pasta da Luna:

cd ~/.luna-kernel
# Modo direto (mostra logs no terminal)
node telegram-luna-adapter.cjs

# Ou via PM2 (background, recomendado)
pm2 start telegram-luna-adapter.cjs --name luna-kernel
pm2 save
pm2 startup
2

Usar no Telegram

Adicione o bot ao seu grupo:

Bot: @lunanexobot

Comandos principais:

  • /kimi — Liga/desliga modo persistente
  • /kimi on — Liga (Luna responde tudo)
  • /kimi off — Desliga (ignora mensagens)
  • /status — Status da Luna

💡 Dicas de uso em grupo:

  • • Quando /kimi on está ativo, QUALQUER mensagem no grupo é processada pela Luna
  • • A Luna executa no PC do Abner — cuidado com comandos destrutivos
  • • Use "Luna, ..." no início da frase para garantir atenção
  • • A Luna lembra contexto entre mensagens no mesmo chat
3

O que a Luna pode fazer?

🛠️ Criar Projetos

"Cria um app React de lista de tarefas" → ela cria arquivos, instala deps, configura Tailwind, tira screenshot

📁 Gerenciar Arquivos

"Leia o arquivo X", "Edite a função Y", "Crie uma pasta Z" — tudo no PC real

🌐 Navegar na Web

"Pesquisa como usar Docker" → ela busca, lê artigos, resume

📊 Dashboard NEXO

"Cria tarefa: revisar site cliente X, prioridade alta" → criada no sistema

🔧 Auto-correção

Se um projeto quebra, ela lê o erro, corrige e testa sozinha

📜 Scripts Workflow

Para tarefas complexas, ela pode executar scripts bash/python/node completos

🔧 Problemas comuns

Timeout 30000ms

Kimi Web demorou. A Luna tenta novamente automaticamente. Verifique se Chrome está na porta 9222.

ERR_ABORTED

Página do Kimi caiu. A Luna limpa páginas antigas e recria. Normalmente resolve sozinho.

npm not found

Node.js não está instalado ou não está no PATH. Reinstale o Node.js v20+.

Bot não responde

Verifique se o TELEGRAM_BOT_TOKEN está correto. Verifique se o bot não foi bloqueado pelo Telegram.
